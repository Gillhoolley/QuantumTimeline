# Quantum Timeline

Interactive standalone webpage showing major quantum computing milestones across a 40+ year history.

## Concept

A horizontal timeline helps beginners understand that quantum computing is not brand new.  
It traces key milestones from foundational theory to current hardware and error-correction progress.

## Included milestones

- 1981 — Feynman proposes quantum simulation
- 1994 — Shor's algorithm
- 2001 — First Shor demonstration
- 2019 — Google quantum supremacy experiment
- 2023 — IBM 1000+ qubits
- 2025+ — Error-corrected prototypes

## Visual categories

- Algorithms
- Hardware
- Government programs
- Major investments

## Features

- Horizontal interactive timeline
- Drag-to-scroll navigation
- Category filter chips
- Milestone cards with tags and descriptions
